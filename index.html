/** * ฟังก์ชันสำหรับรับคำขอแบบ GET (มักจะถูกเรียกเมื่อเปิดลิงก์ผ่านเบราว์เซอร์โดยตรง หรือระบบทดสอบของสคริปต์) */ function doGet(e) { // ส่งข้อความตอบกลับในรูปแบบ JSON เพื่อแจ้งว่าระบบพร้อมใช้งาน return ContentService.createTextOutput(JSON.stringify({ status: "active", message: "ระบบเชื่อมต่อสเปรดชีตพร้อมใช้งานแล้ว (กรุณาส่งข้อมูลผ่านเมธอด POST เท่านั้น)" })) .setMimeType(ContentService.MimeType.JSON); } /** * ฟังก์ชันสำหรับรับข้อมูลที่ส่งผ่านเมธอด POST (จากเว็บระบบรับ-ส่งมอบเครื่องมือ) */ function doPost(e) { try { // 1. ดึงและแปลงข้อมูล JSON ที่ส่งมาจากหน้าเว็บ const data = JSON.parse(e.postData.contents); // 2. เปิดไปยัง Google Sheet ที่ต้องการเก็บข้อมูล // หมายเหตุ: ตรวจสอบให้แน่ใจว่าได้ระบุ ID ของสเปรดชีตถูกต้อง const ss = SpreadsheetApp.openById("1HwSjsZ-SOK75x6-qz9q_CQm_XYkoFpkkT8SJAmbcsQQ"); // ค้นหาชีตที่ชื่อ "Sheet1" หากไม่พบ ให้เลือกชีตแรกแทน const sheet = ss.getSheetByName("Sheet1") || ss.getSheets()[0]; // 3. แนบแถวข้อมูลใหม่ลงในชีตตามลำดับคอลัมน์ sheet.appendRow([ data.date, // วันที่ (บันทึกใหม่) data.time, // เวลา / กะ data.car, // ทะเบียนรถ data.sender, // ผู้ส่งมอบ data.receiver, // ผู้รับมอบ data.qty, // จำนวนเครื่องมือ (ครบ/ไม่ครบ) data.cond, // สภาพเครื่องมือ (พร้อมใช้/ชำรุด) data.condCount || "", // จำนวนที่ชำรุด (ถ้ามี) data.condDetail || "", // รายการชำรุด (ถ้ามี) data.readyCount || "", // จำนวนที่พร้อมใช้ (ถ้ามี) data.note || "", // หมายเหตุ data.sigSender || "", // ภาพลายเซ็นผู้ส่งมอบ (Base64) data.sigReceiver || "", // ภาพลายเซ็นผู้รับมอบ (Base64) new Date().toLocaleString('th-TH') // วันเวลาบันทึกระบบส่วนกลาง (ประทับเวลา) ]); // 4. ส่งผลลัพธ์ตอบกลับในรูปแบบ JSON (Google API จะจัดการ CORS ด้านความปลอดภัยให้เอง) return ContentService.createTextOutput(JSON.stringify({ status: "success", message: "ข้อมูลได้รับการบันทึกเรียบร้อยแล้ว" })) .setMimeType(ContentService.MimeType.JSON); } catch (error) { // กรณีเกิดข้อผิดพลาดในการเขียนสเปรดชีต ให้ส่งกลับข้อมูลข้อผิดพลาด return ContentService.createTextOutput(JSON.stringify({ status: "error", message: error.toString() })) .setMimeType(ContentService.MimeType.JSON); } }
